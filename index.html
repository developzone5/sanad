One task, seven moves

Each step is a skill I reach for on purpose. The rulebook (CLAUDE.md) runs beneath all of them.

02 / SKILLS

The seven in detail

Nothing runs in the background. Each skill loads only when you call it.

/spec

spec

Keeps one living spec file per feature (goal, decisions, plan, risks, log), so a long feature stays on course across many sessions.

Plan
/braintrust

braintrust

For a tough call, it brings in 3 to 5 expert viewpoints (pragmatist, architect, skeptic, performance, product). Each makes its case, then you get one clear recommendation. Runs entirely on Claude, no API keys.

Decide
/gate

gate

Checks the change against my full Definition of Done and security list: builds, tests, security, and edge cases, with every claim backed by real command output.

Verify
/qa

qa

The human-QA gate after /gate: stop being the author, become a skeptical tester in front of the running change. Exploratory user testing, acceptance criteria, usability feel, regression by feel — degrades to API/job/data QA when there's no UI. Gate proves the code is correct; qa judges whether the fix is actually good. Ends in a Ship / Don't ship verdict.

Test
/ship

ship

Writes a plain-language brief of the change (what, why, proof, impact), then opens a draft pull request on a safe branch, never straight to production, and waits for your approval.

Ship
full-output

full-output-enforcement

Guarantees complete, ready-to-paste code: no // ..., no TODOs, no skipped sections, even when the output runs long.

Complete
impeccable

impeccable

Designs and refines interfaces end to end: layout, typography, color, motion, accessibility, and right-to-left support, grounded in a real design system.

Design

04 / GUARDRAILS

The safety net

Always on, predictable, cheap. They stay out of the way until they are needed.

pre-commit

Secrets guard

A lightweight check that runs only on git commit and git push, and blocks it if the change carries a hardcoded secret, a staged .env, or an AI-attribution line in a commit. It is plain script (no AI, no delay), and if it ever errors it lets you through rather than locking up your terminal.

hooks/
Stop hook

End-of-turn nudge

Prints the loop after every turn, so the next move is always one keystroke away: /spec · /braintrust · /gate · /qa · /ship.

settings
CLAUDE.md

The rulebook

Always-on: senior backend bar, the security gate, Definition of Done, git and PR flow, and a hard "be short by default" rule.

global

Take the whole setup

One zip with the rulebook, settings, all seven skills, and the secrets guard. Drop it into your ~/.claude and re-login.
